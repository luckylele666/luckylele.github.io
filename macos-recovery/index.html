通过 u盘启动器恢复并降级 macOS

本文最后更新于：June 18, 2022 am

勿用非正式版系统！

前几天 WWDC 发了各个设备的 OS，这不，iOS 不敢装，就想尝鲜下 macOS 13，这下可好，升级到 Ventura，新的功能——Stage Manager 还没用习惯，就发现 bug 很多：

  • 性能下降，点设置很多时候没反应
  • Typora 用不了（也就写不了这篇文章）
  • Karabiner 改键失效
  • 掉电严重（不能忍）

果然还是不能轻易尝试开发者预览版，不适合我，不给 Apple 当小白鼠，赶紧回退 macOS Monterey 吧。

「降级」的分类

先介绍下传统 Intel Mac 的降级方法：

开机后一直按住

  1. Command + R：Recovery 当前系统

  2. Shift + Option + Command + R：设备出厂版本的 macOS Recovery（或与它最接近且仍在提供的版本）

  3. Option + Command + R：设备兼容的最新版 macOS Recovery

    后两种都需要网络，非常慢，自己要等个三四个小时。跑到 Apple 零售店去让 Genius bar 下载吧[1]，据说每家店都有自己的内网缓存服务器，下载很快，

    结果我这 M1 Pro 芯片的设备还挺麻烦，Apple Silicon 就只剩第一个恢复方法了。

    可能有朋友会说直接用 Time Machine 恢复不就好了，我试过，提示用「迁移助理」传输数据，然后系统直接会提示不支持降级！还有什么 macOS IPSW 软件包，DFU 模式恢复，手头没有多的 Mac 设备暂且不表。

    通过 u盘启动器 Recovery

    找了资料也问了 Apple 客服（说无法降级）。对我来说只有一种方法能救——制作 u盘启动器然后引导恢复系统，Apple 的页面根本就没有贴出 .app 的安装包的下载地址，给的链接会跳转到 Mac App Store，然而会提示不能在新系统安装（就是不让降级），白给。

    把 Monterey OS 的安装包放在这给有需要的朋友。安装这个 pkg 后会释放出一个 .app 格式，插上 u盘，推荐用高性能的 SSD，何谓高性能的 SSD，看我之前的文章。首先把 u盘格式化为 macOS 扩展格式，在终端输入对应的 createinstallmedia 命令，Monterey 如下：

MyVolume 就是外接盘的名字（取名 Fuck Ventura beta）。

  1. 按下 Return 键（回车键）执行命令

  2. 出现 password，输入管理员密码，再次按下 Return 键。键入密码时，「终端」不会显示字符。

  3. 接着键入 Y 确认，再按下 Return 键。

  4. 抹掉数据后，会有提示「终端」要访问可移除宗卷上的文件，点 OK 继续拷贝。

擦除与 u盘的性能有关，这个过程比较漫长……

Apple 芯片的电脑是自动感应开机的（不知道以前的是不是），按任何按键或者开盖都会开机，正确的做法是关机状态下长按 Touch ID 按键直至出现提示（正在载入启动选项），进入 option（引导界面），先擦除主机硬盘上的所有数据（格式化），如果没有这个选项，请返回点其他选项，

文件格式选了 APFS（大小写，加密）安全点嘛，抹掉后就可以安装 macOS Monterey 了，然后就需要耐心的等待。

这里要提醒，保持电源接通，网络稳定，我按到电源按键，网断了，重新安装！安装也取决于 u盘速度，出去骑个车兜风，回来就好了，估计在两个小时左右吧，再用 Time Machine 恢复数据，这个过程挺快，二十分钟，SSD 性能好的优势就来了。

恢复后

恢复系统后可能会有小的毛病，比如图标显示不正常，软件权限（辅助功能里的权限）需要重给，多半「退出登录」一下就好了，重启大法好。Time Machine 神奇的地方在于我当时剪贴板里的东西和打开的网页仍在。

不过恢复后，启动台全部打乱了……灾难啊，直接抛弃了，用 Raycast 这个软件找到快多了，想不起来的软件多半是不会用的。甚至之前 Quantumult X（回退之前的老版本） 的 bug 也没了，所以用着没事，忍住，别瞎升级……

收获

没用的知识有增加了（自嘲一下），遇到麻烦不要急，先 Google，把中文描述转成英文，上 Reddit 等科技宅很多的网站查查，问问懂行的大神，比如前面提到的写 Karabiner 改键的作者，感谢余光😄

我现在挺耐心的，等待恢复的时候去骑个车，就无感了，当然手头有多的设备和 SSD 是最好的，对了给路由器加上代理软件也要提上日程了。

这里有几个疑惑的点：

  1. 之前 Time Machine 备份的数据快塞满一个 500G 的 SSD 了，但是实际数据只有最后一次备份的，这是为什么呢？
  2. 在 Recovery 的过程中若没有 Wi-Fi，可以用网卡连接网络来应用吗？

P.S. 用 Windows 时都没有重装过系统，第一次重装「献给」 macOS……
