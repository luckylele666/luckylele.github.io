单板小白的滑雪初体验

本文最后更新于：January 20, 2023 pm

滑雪对我来说是一种很陌生的运动，以前听亲朋好友说过有室内滑雪，是没有想过会去尝试的运动。没想到前阵子居然有机会去滑雪，带只有一面之缘的我去滑雪，也是一段故事的开始。那就来讲讲我的滑雪体验吧，毕竟只尝试了两次，写错的地方请指出，见谅。

出行

因为雪场在附近城市，交通上还是不方便的，需要开车走高速或者高铁出行，三人成行的话还是开车方便。工作日的下午都有许多人来滑雪，看车牌都是附近城市的，有块室内滑雪天地不容易。

滑雪装备

和骑车一样，通常都是穿三层：

  1. 打底速干层（Base Layer）
  2. 中间保温层（Mid Layer）
  3. 防水防风外壳（Shell）

这里以我去滑的室内场举例子，我觉得穿 Base Layer 加 Shell 就够了，不过据我观察，有些老手，穿 Base Layer 和 Mid Layer（卫衣）的，很 cool 但是冷不冷可能只有他们自己知道吧。

我第一次去滑就是老老实实把三层都穿上。速干衣，可以选择健身房穿的那种，关键是要能透气排汗。中间层选择了羽绒服，结果事实证明我穿多了，很热。可以用抓绒衫/卫衣代替。最外层穿了同行伙伴借给我的 Arc’teryx Alpha SV，轻便帅气最重要，室内雪场没有风，对性能要求不高。

下肢比上肢抗冻，只要里面穿个压缩 legging（不穿都没事，但我喜欢给肌肉一点压迫）加一条滑雪裤/冲锋裤（防水）就好了。

对于新手来说还需要准备几样护具：

  • 外绑护膝
  • 护甲（穿在 Base Layer 外）
  • 护臀（各种好玩可爱的造型，我那天是小怪兽）

至于雪鞋和雪板（单板和双板），在雪场可以租赁，而且包含在雪票里面，无需额外费用。

购票及注意事项

起初我不知道可以在闲鱼上买票，经老鸟指点，原来闲鱼上卖的票要比官方售价便宜 ¥38（周末要贵 ¥50）。买票需要填身份证信息。

进场「照例」扫码，还要再填个人信息（身份证，来自哪里种种）。在大厅用短信息凭证在自助柜机取卡，需要交五百块押金。取卡后就需要更换衣服，雪服和雪裤也是在一楼大厅凭卡租赁。女更衣室在一楼比较方便，男更衣室在负一楼（需要刷卡进入）刷卡后开始计时。雪票是 4 小时，超时需要在押金里扣除的，注意看时间。所以男士们最好在进场前就换好衣服，节约时间。

租一个更衣室的柜子需要十块（押金里面扣），有点坑的，更大的柜子更贵。

挑选和穿戴雪具

进入内场后租赁雪板和雪鞋，雪场提供的雪鞋是最传统的系带式，当笔直站立时，脚趾微微抵住鞋头，但当身体前倾，压迫感消失，这才是合适的 size。把鞋带全部绑紧需要好一会儿功夫（我选择左右脚不同号，不要怕折腾，工作人员会满足你）。

开始滑雪

在穿戴整齐后推开两层大门后就能感受到冷气（没进去前很热），看到大片的白雪就会很兴奋，而且可能是室内的缘故，雪面都没有被踩脏。

推荐做一下热身运动再开始，但是我们直接就上了「魔毯」，「魔毯」是一条长运输带，连人带滑雪板送到顶部，然后滑到底。一般是非常初级的雪道上用的，雪道短，只有 100 米，传输速度很慢。滑雪一分钟，搭「魔毯」四分钟。据我目测，单板玩家的数量占据了九成，可能大家都觉得单板帅一点？

先把雪鞋卡进固定器，雪场的固定器是绑带式固定器（strap-in bindings）。有鞋头和脚踝两根绑带来固定。刚开始看着有坡度的雪道，我还是有点慌的，但是同行的伙伴说不要怕，直接下，给我做了示范动作，我看了好几遍后才敢下去（赶鸭子上架），果然没几米就前扑着地，穿的厚实，有护具，摔的不疼，没有那么慌了。就多扑腾几下，当然还是摔，不过摔多了，胆子大了些，身体也没那么僵硬了。

初学者一般先学习后刃，上身保持放松和直立位，靠双腿来控制速度和刹停。脚后跟用力点地，脚尖指天，越用力，雪板和坡面倾斜角度越大，就越快刹停。刚开始找感觉，要么两只脚发力不均衡，会往一边倾斜，要么就是倾斜角度过大，一屁股坐雪上。对身体的协调能力要求比较高吧，用核心去控制。个人感觉室内滑雪对体能要求并不高，没有让我喘气的感觉。

技术要点

前后刃：

前刃 (Leading Edge)

滑雪时位置相对较前的板刃（脚趾指向的板刃）

后刃 (Tailing Edge)

滑雪时位置相对较后的板刃（脚跟指向的板刃）

我是 Regular 站姿（左脚在雪板前面），Goofy 的站姿就是右脚在前。当两边方向都能掌握就可以尝试前刃了。

我是觉得前刃比较难，对身体的控制力要求更高，和深蹲的发力模式有所不同，不是髋部和膝部同时下降，在保持上身直立的时候膝盖下降，人往前跪，就刹停了。初试前刃有一点恐惧感，因为背部朝向坡底，但是调整好重心就没事，毕竟坡面不陡，不会直接滚下去。

打好基础，循序渐进，不要过快进阶动作。我滑了几趟就盲目的尝试一降到底，结果速度过快差点变成「鱼雷」（新手的贬义代名词，精准制导其他滑雪者，造成危害）撞到其他人。如果撞到后受伤，雪具受损可是很麻烦的事情，要保护自己，保护他人。

自己总结了：身体要放松，手放在身体两侧，不要动，以免影响重心。目光看要去的方向，板子和身体方向指向哪里。肩部、髋部和膝盖同时发力和转向，但是幅度要小。

那么多尝试，就享受白雪世界带来的快乐吧。
