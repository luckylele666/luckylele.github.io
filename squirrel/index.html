2022 Squirrel 鼠须管配置

本文最后更新于：April 5, 2022 pm

痛点

上手了 macOS 也有几周之久了，始终用不习惯原生的中文输入法。不能通过按 Caps Lock 来切换大小写。按住 Shift 输入大写字母在中英文混合输入特别容易误触，比如打 Shift 错打成 SHift，不知道妳有没有这样的体会。

即使在系统偏好设置-键盘-输入法里取消勾选「使用 “中/英” 键切换 “ABC” 输入模式」也是不一样的。按 Caps Lock 不会打出大写字母。

在网上浏览一番寻找解决方法，v2ex 上和我一样习惯的用户不在少数，大家都习惯 Caps Lock 切换大写，通过 Karabiner 改键也无法实现我的需求。最终我需要更换输入法，已经很多年没有使用过收集用户隐私的搜狗输入法了，即使他完美解决我的问题，依然 pass（何况在某些软件环境中会有 glitch）。我把目光投向了「鼠须管」这款输入法。

其特点：

  • 开源、不联网保护隐私、自定义
  • 可使用搜狗词库（带词频）
  • Emoji 与 Mac 同步
  • 动态输入时间、日期、大小写

就是看中了他的第一点，我也得选择接受她的缺点。配置很麻烦，没有图形控制界面，网上有个第三方的图形化配置工具，然而并不好用，还是得自己学习如何配置。

鼠须管 Squirrel 官方安装方式：

Install via Homebrew: brew install --cask squirrel

推荐两个教程

  1. ssnhd/rime（详细教程而且有配置，但是初始词库不好用）

  2. 壹页单章的教程（作者帮我解决了一些问题）

使用注意

使用前要备份，弄错了可以还原，从头来过。每次修改配置后记得 ⌃⌥` 重新部署方能生效。如果配置不生效，那么打开活动监视器把后台杀掉，或者退出登录用户。

使用后有一点不满意的，因为其是不联网非智能输入法，所以需要多用几天培养使用习惯，原生输入法最大的优势在于词库和使用了多年的 iOS 词库相通。

折腾告一段落，在几年前就听说过「鼠须管」这款别具一格的输入法，在这个效率优先的年代选择这么一款古风输入法也能算是有一点 geek 精神吧。
